Isomorphism¶

is_isomorphic(G1: 'Any', G2: 'Any') -> 'bool'¶

Check if two graphs are isomorphic.

Uses a simple degree-sequence + VF2-like backtracking approach for small graphs. This is not as sophisticated as NetworkX's full VF2 implementation but works correctly for the test cases.
